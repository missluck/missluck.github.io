JAVA字符串

不可变字符串String

  • String常用方法

    • int length()返回字符串长度
    • boolean isEmpty()判断字符串长度是否为0，长度为0返回true
    • char charAt(int index)返回指定索引处的字符
    • boolean equals(Object anObject)将此对象与指定的对象比较（通常用来比较两个字符串是否相等）
    • int indexOf(int ch)返回从左向右查找字符的索引
    • int indexOf(String str)返回从左向右查找子字符串的索引
    • int lastIndexOf(int ch)返回从右向左查找的字符索引
    • int lastIndexOf(String str)返回从右向左查找子字符串的索引
    • String substring(int beginIndex)返回一个从索引beginIndex开始截取到最后的字符串
    • String substring(int beginIndex, int endIndex)返回一个从索引beginIndex开始截取到索引endIndex的字符串
    • String[] split(String regex)根据给定正则表达式的匹配拆分此字符串
    • String replace(CharSequence target, CharSequence replacement)根据给定的replacement来替换字符串中的target
    • toLowerCase()将字符串全转换成小写
    • toUpperCase()将字符串全转换成大写
