#JAVA时间类

时间类关系图

  • Java.util.date 核心类
  • Java.text.DateFormat 时间格式化 SimpleDateFormat时间对象和字符串转换
  • Java.util.Calendar日期类
  • Java中时间是从标准纪元1970.1.1 0点开始到某个时刻的毫秒数,类型为long

    Date类常用方法

  • long getTime() 返回从标准纪元1970.1.1 0点开始到Date对象的毫秒数
  • void setTime(long time) 设置Date对象，表示从标准纪元1970.1.1 0点开始以后time毫秒的时间点
  • boolean after(Date when) 测试此日期是否在指定日期之后(是否大于when)
  • boolean before(Date when) 测试此日期是否在指定日期之前(是否小于when)
  • boolean equals(Object obj) 比较两个日期是否相等
  • hashCode() 返回此对象的哈希码值

    DateFormat与SimpleDateFormat

  • 它们的主要工作是完成字符串与时间的转化
  • DateFormat是抽象类
  • SimpleDateFormat是DateFormat的实现类
  • String format(Date date) 传入一个时间对象返回一个时间字符串
  • Date parse(String dateStr) 传入一个字符串返回一个时间对象

  • 字符串与时间相互转换的过程中格式化时间必须一致

    Calendar与GregorianCalendar

  • 负责long类型的数(Date类中的fastTime属性)与日期进行转换 日期计算
  • Calendar是抽象类
  • GregorianCalendar是Calendar的实现类
  • 月份:一月是0,二月是1,以此类推十二月是11
  • 星期:周日是1,周一是2,以此类推周六是7
